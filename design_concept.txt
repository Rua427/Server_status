Dashboard
    : 도넛형 차트로 각 서버별 good, warning, bad, disconnect 개수를 보여준다,.

각 서버종류 별
    : 도넛형 차트로 ram, cpu, disk, cpu temp 4개 박스안에 각 good, warning, bad, disconnect 개수를 보여준다,.
    : 주차별 각 점검 대상의 대한 값을 출력한다.
    : vDU에 경우에는 vnfd, gps를 추가한다.

각 서버 클릭 시
    : n시간마다 업데이트 되며 가로 꺾은선 그래프로 ram, cpu, disk, cpu temp 4개 박스안에 차트를 넣어준다. (평균 값)
    : vDU에 경우에는 vnfd, gps 상태를 일반 텍스트로 현재 상태를 알려준다.



알림창
1. 알림창은 팝업으로 표시하며 활성화 비활성화 가능하다. (on/off)
2. 알림아이콘이 있으며 해당 아이콘 상단에 발생한 알림 개수를 표시한다 ( 빨간색 원형 배경, 흰색 글씨 )
3. 알림창내의 모습은 각 알림이 리스트로 존재한다.
4. 알림창내의 내용은 다음과 같다.
    1. ip정보
    2. 서버 이름
    3. 무엇이 문제인지에 대한 내용 (warning, bad, disconnect) (여러개 일경우 모두 출력)
    4. 알림창내의 내용 박스를 클릭시 chart 출력 영역을 클릭한 서버로 바꾸며 서버 목록에서 해당 서버위치로 스크롤 이동시킨다.
